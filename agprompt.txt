## Contexto
Proyecto: De Clarix - Gestor de boletas para declaracion de impuestos (Chile).
Stack: React + Vite + TypeScript (parcial), Zustand, IndexedDB/Supabase, Recharts, react-router-dom.
NO es e-commerce. Es para que una contadora (hermana del usuario) cargue boletas, las revise y exporte a plantilla Saludent (.xlsm) para F29.

Estado actual: Conectado a Supabase (nube). Datos sincronizados entre local, Vercel y movil.
